🎵 NexusPlay 🎥 🖼️
NexusPlay es un reproductor multimedia integral diseñado específicamente para Windows. Una solución única, ligera y elegante para gestionar y disfrutar de toda tu biblioteca de música, fotos y videos en un solo lugar.

✨ Características Principales
Reproducción Universal: Soporte para los formatos más populares (MP4, MKV, MP3, FLAC, JPG, PNG, etc.).

Interfaz Fluida: Diseñada con estética moderna, optimizada para la experiencia de usuario en Windows.

Biblioteca Inteligente: Organiza automáticamente tu contenido por categorías.

Modo Galería: Visualizador de fotos con soporte para presentaciones de diapositivas.

Rendimiento Optimizado: Bajo consumo de recursos incluso con archivos en alta resolución (4K).
